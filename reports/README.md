# StockGuru — Automated Daily Reports

This folder contains all automated reports generated by your scheduled tasks.
Reports run every day at the times shown below.

---

## 📋 How to Use This Folder

### Quick Check (open these every morning)
| File | What It Tells You |
|------|-------------------|
| `LATEST_channel_debug.md` | Are all APIs and connections working? |
| `LATEST_code_health.md` | Any syntax errors or code issues? |
| `LATEST_test_results.md` | Which tests pass/fail + exact fixes |
| `LATEST_data_integrity.md` | Are data files healthy? Paper portfolio status |
| `LATEST_milestone_tracker.md` | What to build TODAY + launch timeline |

### Daily Archive
The `daily/` subfolder keeps one copy of every report per day, named with the date.
Example: `daily/channel-debug-2026-03-02.md`

## 🚀 Development Workflow

1. **Open Cowork at 9:30 AM** — all 5 reports will be ready
2. **Read LATEST_milestone_tracker.md first** — it tells you the top 3 tasks for today
3. **Check LATEST_channel_debug.md** — fix any broken API connections before coding
4. **Tell Claude** what task to work on — Claude writes the code, you review
5. **End of day** — check LATEST_test_results.md to confirm nothing is broken
